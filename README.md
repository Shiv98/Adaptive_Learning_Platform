## About the project:
 - Database used is sqlite
 - Framework used is Flask
 - Testing is done on Postman

## To Run the Project:
 - Clone the repo
 - Install python 3
 - Install the required packages from pipfile
 - To run the project on localhost, run the following commands on cmd:
    - python 
    - from app import db
    - db.create_all()
    - exit()
    - python app.py
